# 이어줄게

## 3D 체험과 AI 창작으로 전통문화의 단절을 잇다

---

# 1. 발표용 핵심 문장

> **전통문화의 문제는 사라졌다는 것이 아니라, 현대인의 일상적 소비 구조와 단절되어 있다는 것이다.**

> **이어줄게는 이 단절을 3D 체험, AI 창작, 실제 방문 연결로 잇는다.**

> **차별점은 참여다. 대중이 전통문화를 보기만 하는 것이 아니라 직접 만들어본다.**

발표에서는 이 세 문장을 중심으로 말한다. 문제를 많이 늘리지 않고, "단절을 잇는다"는 하나의 구조로 밀고 간다.

---

# 2. 한 줄 정의

> **사용자가 웹 3D 공간에서 전통문화 유산을 체험하고, 그 요소를 AI 이미지·노래·영상으로 창작한 뒤, 실제 문화유산 방문으로 이어주는 전통문화 이어짐 플랫폼.**

---

# 3. 문제 정의

## 루트 원인: 전통문화가 현대 소비구조와 단절되어 있다

해커톤 발제문은 문화를 특정한 행사나 일회적 소비가 아니라, 일상 속에서 지속적으로 경험되는 구조로 전환해야 한다고 말한다.

하지만 전통문화는 아직 현대인의 일상적 소비 구조 안에서 자연스럽게 작동하지 못한다. 박물관과 행사에는 존재하지만, 대중이 반복해서 접하고 직접 참여하는 구조로 이어지지 못한다.

이 하나의 단절이 두 가지 문제로 드러난다.

## 접근의 단절

전통문화 경험은 여전히 물리적 장소에 크게 묶여 있다. 어디에 사는지, 문화시설까지 얼마나 멀리 이동해야 하는지에 따라 경험 기회가 달라진다.

2023년 기준 지역문화실태조사에 따르면, 기초자치단체의 문화예술시설 평균 수는 구 지역 19.7개, 군 지역 7.4개로 차이가 난다. 문화예술시설 평균 접근거리도 구 지역 2.6km, 군 지역 12.6km 수준으로 벌어진다.

> 발표 문장: **"전통문화는 존재하지만, 모두에게 같은 거리로 열려 있지는 않습니다."**

## 소비의 단절

많은 전통문화는 보존의 대상으로는 남아 있지만, 현대 사회 속에서 계속 선택되고 재생산되는 구조는 약하다. 관람하고 끝나는 전시형 소비만으로는 일상 속 반복 경험을 만들기 어렵다.

전통문화의 지속가능성 문제는 단순히 보존하느냐의 문제가 아니다. 현대적 소비 구조 속에서 계속 선택되고, 다시 만들어지고, 공유되는 구조를 갖추었는가의 문제다.

> 발표 문장: **"전통문화는 보존만으로는 이어지지 않습니다. 누군가 다시 경험하고, 만들어보고, 공유해야 이어집니다."**

## 하나의 결론

접근의 단절과 소비의 단절은 따로 떨어진 문제가 아니다.

> **전통문화가 현대인의 일상적 소비 구조와 단절되어 있기 때문에, 접근도 어렵고 소비도 이어지지 않는다.**

그래서 우리의 해결도 하나다.

> **단절된 전통문화를 다시 이어준다.**

---

# 4. 해결 구조: "이어짐"

"이어줄게"는 전통문화와 대중 사이의 단절을 세 단계로 잇는다.

```
[1단계] 접근을 잇다
  → 웹 3D 공간에서 전통문화를 체험
  → 지역·거리·비용 장벽을 낮춤

[2단계] 소비를 잇다
  → 전통 요소를 AI 이미지·노래·영상으로 창작
  → 보는 대상에서 참여하는 콘텐츠로 전환

[3단계] 경험을 잇다
  → QR 코드로 실제 문화유산 방문 정보 제공
  → 디지털 관심을 오프라인 방문으로 연결
```

## Before / After

**기존 방식**
- 박물관이나 행사장에 직접 가야 함
- 전시를 보고 끝남
- 다시 찾아갈 이유가 약함

**이어줄게**
- 웹에서 먼저 3D로 체험함
- AI로 직접 콘텐츠를 만들어봄
- QR을 통해 실제 방문으로 이어짐

> 핵심 차이: **참여**

---

# 5. 사용자 흐름

발표와 시연에서는 이 흐름만 보여준다.

```
1. 사용자가 웹에서 3D 전통문화 공간에 입장한다.
2. 첨성대를 클릭해 선덕여왕과 첨성대 이야기를 본다.
3. 스타일을 선택해 AI 이미지·노래·영상을 만들어본다.
4. 만든 결과물을 게시판에서 확인한다.
5. QR을 통해 실제 첨성대 방문 정보로 이동한다.
```

## 핵심 사용자

전통문화에 깊은 관심이 있던 사람이 아니라, 웹·AI 콘텐츠에는 익숙하지만 전통문화와의 접점은 약한 대중을 우선 대상으로 한다.

발표에서는 타깃을 길게 설명하지 않는다.

> **"전통문화와 멀어진 대중이, 웹에서 먼저 경험하고 직접 참여하게 만드는 서비스입니다."**

---

# 6. 핵심 테마: 첨성대와 선덕여왕

## 왜 첨성대인가

첨성대는 한국인에게 익숙한 대표 문화유산이다. 신라 선덕여왕 때 축조된 것으로 전해지는 천문 관측 유산이며, "하늘을 관측해 사람의 삶과 연결한다"는 상징을 가진다.

## 스토리텔링 연결

> "1400년 전, 선덕여왕은 첨성대로 하늘과 사람을 이어주었습니다. 오늘, 이어줄게는 디지털 기술로 전통문화와 대중을 이어줍니다."

이 문장은 발표에서 그대로 사용할 수 있다.

## AI 콘텐츠 소재

- 별자리와 밤하늘
- 선덕여왕
- 신라의 과학과 문화
- 첨성대의 곡선과 돌 구조
- 경주의 실제 방문 경험

---

# 7. AI의 역할

AI를 어렵게 설명하지 않는다.

> **AI는 전통문화 요소를 이미지·노래·영상으로 바꿔, 사용자가 직접 참여하게 만드는 연결 장치다.**

발표에서는 "AI가 전통문화와 현대 콘텐츠 형식을 이어준다" 정도로만 말한다.

---

# 8. MVP 구현 범위

## 하는 것

1. Three.js로 첨성대 중심 3D 전통문화 공간 구현
2. 별자리/파티클 효과로 천문대 분위기 연출
3. 첨성대 클릭 시 설명 + AI 창작 사이드 패널
4. OpenAI API로 이미지 1건 실시간 생성
5. 사전 제작된 노래 1곡, 영상 1개, 이미지 2개 출력
6. AI 콘텐츠 게시판 구현
7. QR 코드 → Vercel 배포된 오프라인 연결 페이지

## 하지 않는 것

1. 정밀한 문화재 복원
2. 실시간 3D Gaussian Splatting
3. 노래·영상 실시간 API 생성
4. 복잡한 RAG 시스템
5. 사용자 로그인/회원 시스템

## MVP의 핵심 메시지

> "이 MVP는 정밀 복원이나 모든 AI 기능의 실시간 생성을 증명하는 것이 아니다. 전통문화가 **체험 → 창작 → 실제 방문**으로 이어지는 구조를 웹에서 증명하는 데 집중한다."
